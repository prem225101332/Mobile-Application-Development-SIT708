# Travel Companion App - Task 2.1P

## Description
This Travel Companion Android App helps to convert your currencies, fuel effieceny and temperature

## Features in this APP
- Currency conversion (USD, AUD, EUR, JPY, GBP)
- Fuel efficiency conversion (mpg, Km/L, Gallon, Litres, Nautical Mile, Km)
- Temperature conversion (Celsius, Fahrenheit, Kelvin)
- Input validation and error handling

## How to Run
1. Clone the repository
2. Open Task_2.1 folder in Android Studio
3. Run on emulator or physical device

## Technologies Used
- Java
- Android Studio
